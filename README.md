Harmony Heights Band Website

Author

Kamogelo Pienaar

Project Description

This project is a multi-page band website created as part of a web development exercise.
The website represents a fictional band called Harmony Heights and demonstrates the use of HTML, CSS, and JavaScript to create a simple interactive website.

The site includes multiple pages, multimedia content, and a basic contact form to simulate a real band website.

---

Technologies Used

- HTML5
- CSS3
- JavaScript
- SPCK Editor (Mobile Code Editor)
- GitHub (Project Hosting and Version Control)

---

Website Features

Home Page

The home page introduces the band and displays images of band members.

Features:

- Band description
- Band member images
- Navigation menu

---

Music Page

The music page allows visitors to experience the band's content.

Features:

- Audio player
- Video performance
- Multimedia embedding

---

Contact Page

The contact page allows visitors to interact with the band.

Features:

- Tour dates list
- Social media links
- Contact form

How to Run the Project

1. Download or clone the repository.
2. Open the project folder.
3. Open index.html in any web browser.
4. Navigate through the website using the menu links.
